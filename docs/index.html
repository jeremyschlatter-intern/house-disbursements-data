About This Project

What is the Statement of Disbursements?

The Statement of Disbursements of the House is a quarterly report published by the Chief Administrative Officer of the U.S. House of Representatives. It documents every expenditure made by the House, including payments from Members' Representational Allowances (MRA), committee budgets, leadership offices, and administrative operations.

These reports have been published for over 50 years. Since 2009, they have been available as downloadable PDFs from house.gov. Beginning in Q1 2016, the House also publishes the data as CSV files.

Data Coverage

  • 2009–2015 (this dataset): Born-digital PDF reports from house.gov, text-extracted and parsed using the ProPublica/Sunlight Foundation parser. 26 quarters, 2.75 million records.
  • 2016–2025 (this dataset): Official CSV files published by the House Chief Administrative Officer. 40 quarters, 4.3 million records.
  • 2009–2015: Born-digital PDF reports available from house.gov. These can be parsed using the ProPublica/Sunlight Foundation parser.
  • 1970–2008: Scanned PDFs digitized by the Boston Public Library and available on the Internet Archive. OCR quality varies.

Unified Schema

Each record includes:

  • Organization — The office making the disbursement
  • Program — Budget program (e.g., "MEMBERS REPRESENTATIONAL ALLOWANCE")
  • Category — Spending category (Personnel, Travel, Equipment, etc.)
  • Transaction Date / Start Date / End Date
  • Vendor Name — The payee
  • Description — Purpose of the expenditure
  • Amount — Dollar amount

Data Notes

  • The CSV format changed over time: 12 columns (2016–2022) expanding to 18 columns (Q4 2022+), adding organization codes, vendor IDs, and budget object codes.
  • Fiscal year references use "FY" and "LY" (Legislative Year) prefixes in newer data, normalized to plain years in aggregations.
  • Negative amounts represent refunds or adjustments.
  • Only DETAIL records (not SUBTOTAL/GRAND TOTAL rows) are used in analyses.

Prior Work

  • Sunlight Foundation — Original PDF parser and House Staff Directory
  • ProPublica — Maintained parser and published expenditure data
  • Daniel Schuman / Demand Progress — Advocacy for structured disbursement data
  • Congressional Data Coalition — Push for machine-readable congressional data

Technical Details

The pipeline downloads all available CSVs from house.gov, normalizes varying column formats into a unified schema, stores everything in a SQLite database, and generates the JSON data files and CSV exports served by this website. Source code available on GitHub.
